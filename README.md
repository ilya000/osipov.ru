# osipov.ru

Статическая версия старого персонального сайта для публикации на GitHub Pages.

## Публикация на GitHub Pages

1. Создать репозиторий на GitHub, например `osipov.ru`.
2. Загрузить в него файлы из этой папки.
3. В настройках репозитория открыть `Settings -> Pages`.
4. В `Build and deployment` выбрать `Deploy from a branch`.
5. Выбрать ветку `main` и папку `/root`.
6. В `Custom domain` указать `osipov.ru`.

## DNS

Для корневого домена GitHub Pages обычно нужны A-записи на адреса GitHub Pages и CNAME для `www`.
Проверьте актуальные значения в документации GitHub перед изменением DNS.
